# miniguia-estudos-notebooklm
Projeto de estruturação de um Notebook LM para arquitetura de software


O que o projeto faz
  Explora as principais na fonte na internet que tratam dos fundamentos e boas práticas sobre a arquitetura de software
Por que o projeto é útil
  Resume o melhor conteúdo para orientar desenvolvedores e estudantes iniciantes em Sistemas da Informação
  
Como os usuários podem começar a usar o projeto
  Apresentações e mapas mentais

Entrega Final:
Mapa mental e apresentação


Prompts:
Ensine-me a diferença entre arquitetura e design de software.
Quais são os principais requisitos não funcionais em arquitetura?
Converse sobre o que essas fontes dizem de Paradigmas e Estilos, no contexto mais amplo de Arquitetura de Software em 2025.
